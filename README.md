# 🤖 Chatbot de Atención al Cliente con IA

Este proyecto es un asistente virtual inteligente diseñado para automatizar la interacción con clientes. Utiliza inteligencia artificial para comprender consultas en lenguaje natural y gestionar respuestas de manera autónoma.

## 🚀 Características Principales
* **Respuestas con IA:** Integración con la API de **Google Gemini** para generar respuestas fluidas y contextuales.
* **Gestión de Datos:** Conexión con **Supabase** para el almacenamiento y recuperación de información.
* **Atención Automatizada:** Capaz de resolver dudas frecuentes sin intervención humana.
* **Seguridad:** Manejo de API Keys y autenticación segura.

## 🛠️ Tecnologías Usadas
* **Lenguaje:** Python
* **IA:** Google Gemini API
* **Base de Datos:** Supabase
* **Entorno:** VS Code

## 📦 Instalación y Uso

1.  **Clonar el repositorio:**
    ```bash
    git clone <tu-link-del-repositorio>
    ```
2.  **Instalar dependencias:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Configuración:**
    Asegúrate de configurar tus variables de entorno (API Keys de Gemini y Supabase) antes de ejecutar el bot.

4.  **Ejecutar:**
    ```bash
    python main.py
    ```
